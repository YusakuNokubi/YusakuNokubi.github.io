    Java

    １年後期と２年前期の講義で学ぶ．教材は新明解Java入門， Java言語プログラミングレッスン下巻．

    初めてのプログラミング．Javaを通してコンパイルや型，オブジェクト指向や継承，ファイルの扱いやスレッドを学んだ．作成したプログラムは 教材の演習のものであるため，公開はしていない．また，同時に習ったアルゴリズムの実装も行った．具体的には，ナップサック問題や行列計算，ニュートン法や辞書アプリなど．

    C

    ２年後期の講義で学ぶ．教材は新明解Ｃ言語の入門編と実践編．

    ２つ目に学んだプログラミング言語．forやifなどが存在すること，構造体はJavaのクラスのようなものであることから習得は簡単であると感じたが， ポインタの扱い以降は非常に難しくなった．コンピュータアーキテクチャの授業で学んでいるため知識はあるものの，プログラムで実装するのにはなかなか 骨が折れた．作成したプログラムはナップサック問題，データ構造でのリスト，スタックとキューをライブラリを用いずに実装するなど．
